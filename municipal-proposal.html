🎯 企画概要

本企画は、鳥獣被害対策の現場課題を、AI×IoT技術と地域の若者の力で解決する共創型ハッカソンです。

Metagri研究所が2025年に実施した「農業AIハッカソン2025」の成功モデルを活用し、自治体の鳥獣被害対策部門が抱える課題を、技術的に解決可能なプロトタイプに変換。さらに社会実装まで伴走支援します。

3つの特徴

💡

① 非エンジニアでも参加可能
「Vibe Coding」推奨

生成AIとの対話でコードを生成する手法により、プログラミング経験のない方でも参加可能。地域の多様な人材が活躍できます。

⚡

② 2週間で実務活用できる成果物

過去ハッカソンでは、受賞作品が2週間で農家の業務で活用開始。プロトタイプで終わらせず、実際に使える解決策を生み出します。

💰

③ 初期費用負担なしで実施可能

自治体の金銭的負担を最小化。実証フィールドと課題提供のみで、技術支援・メンター派遣・運営はMetagri研究所が担当します。

📊 背景・課題認識

自治体が抱える鳥獣被害対策の3大課題

1. 人手不足と高齢化

  • • 猟師・捕獲従事者の高齢化が進行
  • • わなの見回りに膨大な時間がかかる（1日に何度も山林を巡回）
  • • 若手の担い手が不足

2. 効果的な対策の不足

  • • 鳥獣の出没パターンが把握できない
  • • 捕獲情報が紙ベース・属人化しており、データ活用できない
  • • 被害状況と捕獲活動の連携が取れていない

3. 予算制約

  • • 高額なIoT・AIシステムの導入は難しい
  • • 専門業者への外注費用が高額
  • • 費用対効果が見えにくい

なぜハッカソンなのか？

従来の「業者への発注」ではなく、地域の若者が主体的に課題解決に取り組むことで：

✓

低コストで実用的なプロトタイプが生まれる

✓

若者やAIを使える未来のクリエイターの地域定着・UIターン促進につながる

✓

自治体職員・猟師・クリエイターの協働が生まれる

✓

継続的な改善・運用体制ができる

💡 想定される開発テーマ例

以下のような、現場で本当に困っている課題を題材にします。

📷

テーマ1

わな見回り効率化システム

  • • AIカメラでわな内の鳥獣を自動検知
  • • 捕獲時に猟師のスマホへ自動通知
🗺️

テーマ2

鳥獣出没予測マップ

  • • 過去の出没データ・被害データをAI分析
  • • 「明日この地域に出没しやすい」を予測
  • • 農家への事前警戒情報配信
📱

テーマ3

被害報告DXアプリ

  • • 農家がスマホで被害写真を撮影→即報告
  • • AIが被害状況（鳥獣種別・被害レベル）を自動判定
  • • 自治体の集計作業を自動化

💬 課題は自治体と協議して決定

実際の課題提供者（自治体職員・猟師・農家）の「困りごと」をヒアリングし、技術的に解決可能なテーマに落とし込みます。

📅 実施スケジュール（想定）

Phase 1: 企画準備期間（1ヶ月）

Week 1-2: 課題ヒアリング

  • • 自治体担当者・猟師・農家へのヒアリング
  • • 現場視察・データ確認
  • • 解決すべき優先課題の特定

Week 3-4: ハッカソン設計

  • • 開発テーマの確定
  • • 参加者募集開始（地域の高校・大学・企業へ告知）
  • • メンター・審査員の確保

Phase 2: ハッカソン実施（1〜2ヶ月）

Day 1: キックオフ（オンライン or 現地）

  • • 課題提供者からの説明
  • • チームビルディング
  • • 技術レクチャー（生成AI活用法・Vibe Coding）

Week 1-3: 開発期間

  • • チームごとにプロトタイプ開発
  • • Discord上でメンター常時サポート
  • • 週1回のオンライン進捗共有会

Week 4: 最終発表会

  • • 各チームが成果をプレゼン
  • • 課題提供者・審査員による評価
  • • 最優秀賞・優秀賞の選出

Phase 3: 社会実装支援（1ヶ月〜）

受賞作品の実証実験

  • • 課題提供者が実際の現場で使用
  • • フィードバック収集・改善

継続開発支援

  • • 必要に応じてメンター派遣継続
  • • 実装に向けた技術支援

事業化・予算化支援

  • • 自治体の次年度予算への組み込み支援
  • • 他地域への横展開サポート
